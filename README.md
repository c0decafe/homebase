Nix-powered DevContainer for Codespaces, local development, and CI.  
Prebuilt image: `ghcr.io/c0decafe/homebase:latest`.

## Highlights
- DevContainer-first; Nix for everything else
- Exact tool paths generated by the flake and copied into VS Code on startup
- GH Actions builds/pushes the image to GHCR

## Structure
- `.devcontainer/devcontainer.json` + `bootstrap.sh`
- `flake.nix`
- `.github/workflows/container.yml`

## Use
Open in Codespaces (or locally):
```
direnv allow
nix develop
```
